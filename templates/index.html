TIVA APIs

Following are the TIVA APIs:

  • Start an Order: http://smartcart.thecodershub.com/order/make_order/<vendor_id>/
    Returns {<order_id>}.
  • Add an item: http://smartcart.thecodershub.com/order/add_item/<order_id>/<item_rf_id>/
    Returns {<item_name> Rs. <item_price>/-}.
  • Verify weight: http://smartcart.thecodershub.com/order/verify_weight/<order_id>/<weight>/
    Set cart_weight to <weight> and returns {1} if it matches the sum, {0} otherwise.
